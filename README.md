# Core C++ Systems Projects

This repository contains lightweight, optimized C++ command-line utilities focusing on memory efficiency, deterministic data layouts, and algorithmic processing.

## 1. Tournament Bracket & Team Seed Manager
* **Features:** Implements a dynamic single-elimination tournament driver using multi-dimensional Queue arrays.
* **Logic:** Employs a probabilistic match simulation engine weighted by team seeding and performance fatigue metrics.
* **Optimization:** Utilizes fixed-size structures to minimize heap fragmentation during real-time bracket adjustments.

## 2. Materials Inventory & Quality Control System
* **Features:** Logs structural parameters (mass densities, defect counts) for manufacturing batch tracking.
* **Logic:** Automates component filtering against industrial tolerance limits.
* **Optimization:** Implements strict manual memory management via raw pointer arrays and a custom destructor tracking mechanism to maintain a zero-leak memory profile under test cycles.
